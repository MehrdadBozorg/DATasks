1. Introduction:
This text document together with a file containing the data for the data analytics task.

2. Description of the data Set:

Attached data set represents direct mailing data advertisements. The data are described by 9 features and a target column. 
The input attributes are:
"name", "age", "lifestyle", "zip code", "family status", "car", "sports", "earnings", "Living area" .
The target attribute is "label" .

3. Description of the problem:

Goals of this analysis task:
1. to build a data driven model that reads input variables (defined by the 9 features listed above), which characterize a
new customer and predicts his response.
2. is it possible to get an ideal environment (described by the 9 input features) of a customer 
which will mostly respond ?
